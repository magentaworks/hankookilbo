# 새로운 문서 추가
홈 페이지를 제외한 모든 문서는 docs 폴더 아래에 위치합니다.

하위 페이지를 가지지 않는 경우에는 docs 폴더 바로 아래에 md파일을 생성하면 되고 하위 페이지를 가진다면 docs 폴더 아래에 하위 폴더를 하나 만들어서 폴더 이름과 동일한 md 파일을 생성하시면 됩니다.
